## How to re-run on latest versions (data generation)

Run from WSL (repo root):

```bash
./scripts/run-rq1-latest-5-wsl.sh gpt4omini_run3_latest5
```

This script downloads each latest package via `npm pack`, extracts it under `outputs/packages_latest/`, installs dependencies, builds if needed, and runs TestPilot, writing results to:

- `outputs/runs/gpt4omini_run3_latest5/<package>/...`

## Comparison plan (what to compare for RQ1)

RQ1 in this artifact is **“Correlation between function size and function statement coverage.”**

For each package, the minimum comparable summary we can extract from `report.json` is:

- total statement coverage (`coverage.total.statements.pct`)
- total branch coverage (`coverage.total.branches.pct`)
- number of generated tests (`stats.nrTests`, `stats.nrPasses`, `stats.nrFailures`)

To reproduce the paper’s RQ1 statistic exactly (Pearson correlation), it's needed to be run through the scripts in original `artifacts/`
